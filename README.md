# StudyFlow AI Dashboard 🚀
StudyFlow AI Dashboard is a smart student productivity web application designed to help students manage studies more efficiently through progress tracking, assignment management, analytics, and an integrated AI assistant.

## Features ✨
- Subject Progress Tracking
- Study Session Timer
- Assignment & Task Management
- Productivity Analytics
- AI Assistant Chat Section
- Modern Dashboard UI

## Built With 🛠️
- HTML5
- CSS3
- JavaScript

## Purpose 🎯
This project was created to improve frontend development skills by building a complete dashboard-style web application with interactive features and clean UI design.

## Note
The AI Assistant feature requires a valid API key configuration to function properly.
